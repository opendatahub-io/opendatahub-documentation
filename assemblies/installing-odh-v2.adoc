= Installing Open Data Hub (v2)

[role='_abstract']
This section describes how to install the Open Data Hub Operator v2.X on your OpenShift Container Platform from the OpenShift web console. 
//For information about upgrading the Open Hub Operator v1.X to v2.X, see

Installing Open Data Hub involves the following tasks:

. Install the Open Data Hub Operator.
. Install Open Data Hub components.
. Access the Open Data Hub dashboard.

[NOTE]
--
Open Data Hub Operator version 2.X represents an alpha release, only accessible on the *fast* channel. Later releases will change to the *rolling* channel when the operator attains greater stability.
--
