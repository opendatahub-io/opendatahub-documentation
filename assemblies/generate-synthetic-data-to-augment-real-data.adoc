= Generate synthetic data to augment real data

When you customize a model with your own data, you can generate high-quality synthetic data to augment your dataset, improve model robustness, and cover edge cases.

{org-name} provides the Synthetic Data Generation (SDG) Hub, a modular Python framework for building synthetic data generation pipelines by using composable blocks and flows. Each block performs a specific task, such as LLM chat, parse text, evaluate, or transform data. Flows chain blocks together to create complex data generation pipelines that include validation and parameter management. A flow (data generation pipeline) is a YAML specification that defines an instance of a data generation algorithm.
