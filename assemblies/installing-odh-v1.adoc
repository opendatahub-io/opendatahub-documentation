= Installing Open Data Hub (v1)

[role='_abstract']
You can install the Open Data Hub Operator version 1 on your OpenShift Container Platform from the OpenShift web console. For information about installing the Open Hub Operator version 2, see link:{odhdocshome}/installing-odh/#installing-odh-v2[Installing Open Data Hub (v2)].

Installing Open Data Hub involves the following tasks:

. Install the Open Data Hub Operator.
. Create a new project for your Open Data Hub instance.
. Add an Open Data Hub instance.
. Access the Open Data Hub dashboard.
