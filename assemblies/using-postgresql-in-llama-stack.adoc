= Using PostgreSQL in Llama Stack

[role="_abstract"]
PostgreSQL can support Llama Stack deployments in multiple ways. You can use PostgreSQL with the pgvector extension as a remote vector database provider. You can also use PostgreSQL as a metadata storage backend for some vector storage configurations.
