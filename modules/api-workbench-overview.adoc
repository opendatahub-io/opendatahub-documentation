= Overview

[role="_abstract"]
In this user guide, you create a Workbench by using the {productname-long} Custom Resource Definitions (CRDs) and the OpenShift Command Line Interface (CLI). When you create a Workbench, you configure a Notebook image, along with other configurations. The Notebook image is optimized with the tools and libraries that you need for developing and training models.

You can also use the OpenShift APIs to create the custom resources such as the `ImageStream` object and the Workbench object. You can programmatically call the APIs through HTTP GET methods in your code, or a Bash, or a Python script. Refer to the link:https://docs.redhat.com/en/documentation/openshift_container_platform/{ocp-latest-version}/html/image_apis/image-apis#imagestream-image-openshift-io-v1-1[link^] to see how you can use the OpenShift API to create an `ImageStream` resource.
