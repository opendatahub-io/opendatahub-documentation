= Mirror the Python Index for your disconnected environment

If you are using a disconnected environment, use the following code example to access the {org-name} Python index content and copy it locally. You can then upload the packages into your own internal hosting service: 

[source, bash]
----
#!/bin/bash -x

URL=https://console.redhat.com/api/pypi/public-rhai/rhoai/3.0/cuda-ubi9-test/simple/

wget \
--verbose \
--mirror \
--continue \
--no-host-directories \
--cut-dirs=4 \
$URL
----
