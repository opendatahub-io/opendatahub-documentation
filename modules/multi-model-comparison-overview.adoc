= How multi-model comparison accelerates model selection

[role="_abstract"]
You can test many AI models side-by-side with the same prompt by using multi-model comparison.
Comparing model responses helps you identify the best-performing model for your use case based on quality, latency, and behavior.

Multi-model comparison has the following benefits:

* Accelerates model selection by evaluating many models in a single session instead of configuring separate playground instances for each model.
* Reveals performance differences such as variations in response quality, tone, verbosity, and accuracy across models.
* Allows testing how different models respond to the same prompt, helping you select models that are robust to prompt variations.
* Helps compare which models generate responses faster and which models handle specific tasks more effectively.

Multi-model comparison is useful when you need to:

* Evaluate open source models against commercial models.
* Compare different versions of the same model.
* Test how models respond to complex or ambiguous prompts.
* Assess cost-performance tradeoffs for production deployment.
* Select a model for a specific domain.
